## How to Use
 To start the server, enter the default gulp command:
 >$ gulp

**List of gulp commands:**
| Command        | Description  | Notes |
| ------------- |:-------------:|:--: |
| *$ gulp*      | Default task | Makes initial function calls to start server. |
| *$ connect*   | Connects to *localhost:1820*  | |
| *$ watch* 	| Watches files.  |
| *$ rel*       | Task to *rel*oad files. | For gulp's livereload feature. |
| *$ sass*      | Uses Sass to convert .scss to .css | |t
| *$ lint*      | *jshint* task. Uses jshint on .js files. | Uses *jshint-stylish* reporter. |
*(for more info, use $gulp help)*
